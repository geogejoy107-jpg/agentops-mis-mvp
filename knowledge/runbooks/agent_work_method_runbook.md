# Agent Work Method Runbook

1. Read `PROJECT_SPEC.md`, `AGENT_WORKFLOW.md`, and task acceptance criteria.
2. Run `agentops knowledge search "<task keywords>" --refresh`.
3. Inspect relevant base notes from `knowledge/bases/`.
4. Submit `agentops agent-plan create ...`.
5. Execute through Agent Gateway CLI/API.
6. Run focused verification.
7. Record artifact/evaluation/audit evidence and propose memory candidates only for durable lessons.
